# decode-task-4-general-knowledge-quiz

## Description
A simple Python quiz game that asks the user three general knowledge questions and calculates a final score.

## Features
- Multiple-choice style quiz
- Score tracking
- Immediate feedback for answers
- Final score display

## Technologies Used
- Python

## Example

Question: What is the capital of France?

Answer: Paris

Result: Correct!

Final Score: 3/3
